Question-1: What are the different ways to select an element in the DOM?

Different ways to select an element in the DOM :
1) document.getElementById 2) document.getElementsByClassName 3) document.getElementsByTagName 4) document.querySelector 5) document.querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML will take all HTML data inside a div.
innerText will take texts from inside an element/tag.
textContent wil take text content from a node.

Question-4: What is event bubbling in the DOM?

Event keeps looking for elements in the childs of a node, when number of child is finished it returns to its parent. That returning process is called Event bubbling.
